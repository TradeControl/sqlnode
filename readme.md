# ⚠️ Deprecation Notice

This repository is **deprecated** and retained for provenance and historical reference only.

All future development, schema changes, and configuration tooling are now managed in the [tradecontrol.web](https://github.com/TradeControl/tradecontrol.web) repository.

## Where to Find the Latest Work

- **Active schema:** `Schema/tcNodeDb4` in the main repository.
- **Legacy schema and tools:** Available in this repository.
- **Migration mapping:** Documentation and mapping between `tcNodeDb3` and `tcNodeDb4` are provided in the main repository.

## Guidance

- All new implementations and upgrades will use the resources in [tradecontrol.web](https://github.com/TradeControl/tradecontrol.web).
- This repository will no longer receive updates or support.

For migration help or more information, please refer to the documentation in the main repository or open an issue there.
